# SplitGroup

Aplicación para gestionar gastos compartidos entre grupos.

## Requisitos previos

Necesitas tener instalado **Node.js** en tu ordenador.  
Descárgalo aquí: https://nodejs.org (versión LTS)

## Instalación y arranque

### Opción fácil (Windows)
Haz doble clic en el archivo `instalar.bat`

### Opción manual
Abre una terminal en esta carpeta y ejecuta:

- npm install --legacy-peer-deps
- ng serve -o

Luego abre el navegador en: http://localhost:4200

## Si npm no funciona en PowerShell
Abre PowerShell como administrador y ejecuta:

Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned

Luego vuelve a intentarlo.
